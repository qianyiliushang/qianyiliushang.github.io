CentOS 7 安装jira

发表于 2018-01-09   |  

之前一直用的Jira6.3.6，基本上安装一次就不用再动了，现在要用到jira agile中的项目管理，以前安装的时候没有选，所以只能重新安装了，截止目前，jira最新版本是7.6，去官网看了一下，页面UI好看了很多，那就装最新的吧。

依赖

  1. java
  2. mysql

系统要求

  • CPU：Quad core 2GHz+ CPU
  • RAM: 6GB
  • Minimum database space: 10GB
